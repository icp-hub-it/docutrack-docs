# delete_file

Deletes a file from the user's storage canister.

**Arguments:**

- `file_id: nat64` — The ID of the file to delete.

**Returns:**

- `DeleteFileResponse` — Result of the delete operation.

### Candid Definition

```candid
delete_file : (nat64) -> (DeleteFileResponse);
```
