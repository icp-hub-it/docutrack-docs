# get_shared_files

Returns a list of files shared with a specific user.

**Arguments:**

- `user_id: principal` — The user ID to retrieve shares for.

**Returns:**

- `vec PublicFileMetadata` — List of shared file metadata.

### Candid Definition

```candid
get_shared_files : (principal) -> (vec PublicFileMetadata) query;
```
