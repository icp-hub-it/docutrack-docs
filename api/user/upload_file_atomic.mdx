# upload_file_atomic

Atomically uploads an entire file to the user's canister.

**Arguments:**

- `request: UploadFileAtomicRequest` — File data and metadata.

**Returns:**

- `UploadFileAtomicResponse` — Atomic upload result.

### Candid Definition

```candid
upload_file_atomic : (UploadFileAtomicRequest) -> (UploadFileAtomicResponse);
```
