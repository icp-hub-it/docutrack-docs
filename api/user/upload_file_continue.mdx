# upload_file_continue

Uploads subsequent chunks of a file after initiation.

**Arguments:**

- `request: UploadFileContinueRequest` — File chunk and metadata.

**Returns:**

- `UploadFileContinueResponse` — Indicates continuation result.

### Candid Definition

```candid
upload_file_continue : (UploadFileContinueRequest) -> (UploadFileContinueResponse);
```
