# share_file

Shares a file with a specific user.

**Arguments:**

- `user_id: principal` — The user ID to share with.
- `file_id: nat64` — The ID of the file to share.
- `blob: blob` — File key encrypted with the user's public key.

**Returns:**

- `FileSharingResponse` — Indicates share result.

### Candid Definition

```candid
share_file : (principal, nat64, blob) -> (FileSharingResponse);
```
