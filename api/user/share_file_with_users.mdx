# share_file_with_users

Shares a file with multiple users.

**Arguments:**

- `user_ids: vec principal` — List of user IDs to share with.
- `file_id: nat64` — The ID of the file to share.
- `keys: vec blob` — File keys encrypted per user.

### Candid Definition

```candid
share_file_with_users : (vec principal, nat64, vec blob) -> ();
```
