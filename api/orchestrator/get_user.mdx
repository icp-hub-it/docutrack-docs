# get_user

Returns the public information of a user by their user ID.

**Arguments:**

- `user_id: principal` — The user ID of the user to retrieve.

**Returns:**

- `opt PublicUser` — Optional `PublicUser` object containing the user's public information, or null if the user does not exist.

### Candid Definition

```candid
get_user : (principal) -> (opt PublicUser) query;
```
