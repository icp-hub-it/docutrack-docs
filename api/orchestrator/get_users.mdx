# get_users

Returns a paginated list of users.

**Arguments:**

- `pagination: Pagination` — The pagination parameters for the query.
- `search: opt text` — Optional username filter.

**Returns:**

- `GetUsersResponse` — Response object containing users list and pagination info.

### Candid Definition

```candid
get_users : (Pagination, opt text) -> (GetUsersResponse) query;
```
