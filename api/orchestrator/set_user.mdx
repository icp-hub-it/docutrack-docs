# set_user

Signs up with Internet Identity by setting a username. Triggers orchestrator to create user canister.

**Arguments:**

- `username: text` — The desired username.
- `blob: blob` — Blob containing the user's public key.

**Returns:**

- `SetUserResponse` — Indicates the result of user creation.

### Candid Definition

```candid
set_user : (text, blob) -> (SetUserResponse);
```
