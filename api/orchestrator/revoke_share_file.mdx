# revoke_share_file

Revokes access to a shared file for a specific user. Can only be called by the user canister.

**Arguments:**

- `user_id: principal` — The user ID to revoke access from.
- `file_id: nat64` — The ID of the file to revoke.

**Returns:**

- `RevokeShareFileResponse` — Indicates the result of the revocation.

### Candid Definition

```candid
revoke_share_file : (principal, nat64) -> (RevokeShareFileResponse);
```
