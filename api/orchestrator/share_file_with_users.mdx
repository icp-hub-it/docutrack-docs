# share_file_with_users

Shares a file with multiple users. Can only be called by the user canister.

**Arguments:**

- `user_ids: vec principal` — List of user IDs to share with.
- `file_id: nat64` — The ID of the file to share.
- `metadata: ShareFileMetadata` — Metadata about the file.

**Returns:**

- `ShareFileResponse` — Indicates the result of the share.

### Candid Definition

```candid
share_file_with_users : (vec principal, nat64, ShareFileMetadata) -> (ShareFileResponse);
```
