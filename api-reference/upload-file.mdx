# `upload_file`

Uploads a new file to the user’s personal canister.

## Signature

```candid
upload_file : (file_data: FileData) -> async Result
```

## Parameters

- `file_data`: object with:
  - `name: Text`
  - `content_type: Text`
  - `chunks: [Blob]`

## Returns

- `Ok(FileId)` – Unique identifier of the uploaded file.
- `Err(Error)` – Upload failed.
