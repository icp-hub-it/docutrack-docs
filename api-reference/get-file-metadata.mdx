# `get_file_metadata`

Fetches metadata of a file.

## Signature

```candid
get_file_metadata : (file_id: Text) -> async Result
```

## Parameters

- `file_id`: Text – The identifier of the file.

## Returns

- `Ok(FileMetadata)` – Metadata of the file.
- `Err(Error)` – Retrieval failed.
