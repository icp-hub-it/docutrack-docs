# `delete_file`

Deletes a file permanently from the canister.

## Signature

```candid
delete_file : (file_id: Text) -> async Result
```

## Parameters

- `file_id`: Text – The identifier of the file to delete.

## Returns

- `Ok()` – File deleted successfully.
- `Err(Error)` – Deletion failed.
