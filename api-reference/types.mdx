# Type Definitions

## Result

```motoko
variant {
  Ok : any;
  Err : Error;
}
```

## Error

```motoko
variant {
  NotFound;
  Unauthorized;
  InternalError;
  InvalidData;
}
```

## FileData

```motoko
record {
  name : Text;
  content_type : Text;
  chunks : [Blob];
}
```

## FileMetadata

```motoko
record {
  id : Text;
  name : Text;
  content_type : Text;
  size : Nat;
  created_at : Timestamp;
  updated_at : Timestamp;
}
```

## ShareRequest

```motoko
record {
  file_id : Text;
  recipient_principal : Principal;
  permissions : [Permission];
}
```

## Permission

```motoko
variant {
  Read;
  Write;
  Owner;
}
```
