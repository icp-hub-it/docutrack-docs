# `share_file`

Shares a file with another user.

## Signature

```candid
share_file : (share_request: ShareRequest) -> async Result
```

## Parameters

- `share_request`: object with:
  - `file_id: Text`
  - `recipient_principal: Principal`
  - `permissions: [Permission]`

## Returns

- `Ok()` – File successfully shared.
- `Err(Error)` – Sharing failed.
